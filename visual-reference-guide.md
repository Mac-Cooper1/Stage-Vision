## Summary: Key Visual Patterns

### Most Common Furniture
1. Curved white/cream bouclé sofas
2. Light oak/natural wood dining tables
3. Wire-frame or organic wood dining chairs
4. Woven counter/bar stools
5. Sculptural coffee tables

### Most Common Decor
1. Pink protea flowers (3+ images)
2. Large indoor trees/plants (3+ images)
3. Art books on coffee tables
4. Artisanal ceramics
5. Woven textures (rugs, baskets, chairs)

### Most Common Lighting Fixtures
1. Ceramic pendants (perforated, conical)
2. Brass disc/saucer pendants
3. Glass dome pendants
4. Industrial track lighting
5. Sculptural floor lamps

### Most Common Architectural Features
1. Wood beam ceilings (exposed, natural finish)
2. Floor-to-ceiling windows
3. Plaster/stucco walls
4. Indoor-outdoor flow (sliding glass)
5. Views as focal points

### Lighting Timing
- Dusk/golden hour: 10+ images
- Soft daylight: 6+ images
- Interior glow through windows: 8+ images

## Prompt Writing Reference

When writing staging prompts, reference these specific combinations:

**Living Room Template:**
"Curved white bouclé sofa, organic [walnut/woven/glass] coffee table with art books, [cream swivel chairs / leather lounge chairs], large [ficus/olive] tree in [ceramic/concrete] planter, [brown geometric / cream shag / natural fiber] area rug, [brass floor lamp / sculptural table lamp], pink protea in ceramic vase"

**Dining Room Template:**
"Light oak dining table, [wire-frame / organic wood curved-back] chairs, [ceramic perforated / brass disc] pendant light, [single branch in vase / empty], natural fiber rug"

**Kitchen Template:**
"Woven [leather/rattan] counter stools with [black/brass] metal bases, [wood cutting board, ceramic fruit bowl, small herb plant], brass fixtures"

**Lighting Description Template:**
"Warm golden-hour lighting washing through space, soft directional shadows, [balanced exposure with window views / warm interior glow visible]"
